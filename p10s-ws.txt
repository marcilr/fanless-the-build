ASUS P10S WS motherboard


cpu:     Build in Intel Socket 1151 Core i7/Core i5/Core i3/
         for Xeon E3-1200 v5 and 6th genereation Core,
         Pentium and Celeron Processors 

chipset: Intel 236

ram:     4x DIMM, Max. 64GB, DDR4 2133/2400 MHz ECC, Non-ECC UDIMM,
         Un-buffered Memory
         Dual Channel Memory Architecture 
         ** Refer to www.asus.com for the Memory QVL
         (Qualified Vendors Lists).
         --www.asus.com/us/Motherboards/P10S-WS/specifications/

sata:    8x SATA 6Gb/s ports
M.2:     2x M.2 socket 22110

lan:     2x Intel 

video:   HDMI, DisplayPort, DVI-D, and VGA
         Integrated Graphics Processor 
         Multi-VGA output support : HDMI/DVI-D/DisplayPort ports 
         VGA port
         - Supports VGA with max. resolution 1920 x 1200 @ 60 Hz
         - Supports HDMI with max. resolution 4096 x 2160 @ 60 Hz / 
           4096 x 2160 @ 24 Hz
         - Supports DVI-D with max. resolution 1920 x 1200 @ 60 Hz
         - Supports DisplayPort with max. resolution 4096 x 2304 @ 60 Hz 
         Maximum shared memory of 512 MB
         Supports Intel HD Graphics, InTru 3D, Quick Sync Video, 
         Clear Video HD Technology, Insider™
         --www.asus.com/us/Motherboards/P10S-WS/specifications/

         "...you’ll be able to connect and drive three
         independent displays simultaneously – great
         for multitasking, and even better for productivity."
         --www.asus.com/us/Motherboards/P10S-WS/

audio:   Realtek ALC1150 

usb:     4x USB 3.0
         1x USB 3.1 Type A
         1x USB 3.1 Type C

slots:   1 x PCIe 3.0/2.0 x16 (x16/x8)
         1 x PCIe 3.0/2.0 x16 (x8 mode) *1
         1 x PCIe 3.0/2.0 x16 (x4 mode) *2
         1 x PCIe 3.0/2.0 x16 (x4 mode) *2
         --www.asus.com/us/Motherboards/P10S-WS/specifications/
